<h1 align="center" style="font-weight: bold;">Flutter Expenses</h1>
<p align="center">
<b>Flutter Expenses</b> is a personal finance tracker built with Flutter to log and visualize daily expenses with weekly charts.
</p>

## 📍 Features

- **Add Expenses**: Log daily expenses with title, amount and date.
- **Chart Visualization**: Weekly bar chart showing spending distribution.
- **Delete Transactions**: Remove expenses from the list with a swipe.
- **Responsive Layout**: Adapts to both portrait and landscape orientations.

## 💻 Technologies

- Flutter / Dart
- intl

## 💡 How to Use

1. Clone the repository:
```bash
   git clone https://github.com/dm-moro/FlutterExpenses.git
```

2. Navigate to the project directory:
```bash
   cd FlutterExpenses
```

3. Install dependencies:
```bash
   flutter pub get
```

4. Run the app:
```bash
   flutter run
```
